NOTE: If combined catchments or Catchment Discharge are used, then MIKE 1D creates several catchments based on the name of the combined catchment. The names of these extra catchments will be the name of the combined catchment with the type of the component appended. For example, if a combined catchment with a Time Area component is named "Catchment1" then the time area component will be named "Catchment1 - Time-Area". Catchment Discharge components will the the name "Catchment1CatchmentDischarge".
